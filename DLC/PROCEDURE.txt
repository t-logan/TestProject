XML vs. HDF5 Run Procedure
==========================

1) Empty the c:\tmp directory

2) Delete all entries in the stats table in the DLC database

3) Configure the GenerateCSVFiles program arguments to control the number of files, emissions samples, 
   and photo entries generated by the run.

Example:
10 10000 C:\Users\tlogan\Documents\GitHub\TestProject\DLC\src\main\dat\table_04_43.csv C:/tmp/ 100
   ^---- samples ceiling                                                                       ^--- photo ceiling
^---- number of files

4) Run GenerateCSVFiles.java

5) Run WriteXMLFiles.groovy

6) Run WriteHDF5Files.py

7) Run GetFileSizes.java

8) Run ReadXMLFiles.groovy

9) Run ReadHDF5Files.py

10) Run StatsToCSVFile.java

The results are stored in c:\tmp\stats.csv
